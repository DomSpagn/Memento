Memento — Note di rilascio

Cronologia completa di funzionalità e correzioni

v1.1 Marzo 2026
✦ Nuove funzionalità
  • Gli aggiornamenti della cronologia si aggiungono ora tramite il pulsante + Nuovo aggiornamento anziché tramite un campo di testo sempre visibile.
  • La barra di formattazione (Grassetto, Corsivo, Sottolineato, Elenco puntato, Elenco numerato, Colore testo, Citazione, Rimuovi citazione) compare immediatamente quando si crea o si modifica una voce della cronologia.
  • La sezione tag e il pulsante Allega compaiono direttamente dopo aver premuto + Nuovo aggiornamento, senza dover digitare nulla.
  • Ogni voce esistente della cronologia dispone ora di una propria barra di formattazione, sezione tag e pulsante Allega, visibili quando la voce viene messa in modalità modifica.
  • Gli allegati aggiunti durante la composizione di un nuovo aggiornamento vengono salvati atomicamente insieme alla voce.
  • Pagina Note di rilascio (questo file) accessibile dal menu Aiuto.
✦ Correzioni
  • Risolto crash ("str object is not callable") quando si aggiungeva un tag tramite il pulsante + — la variabile del ciclo nascondeva la funzione di traduzione.
  • La DataTable si estende ora per occupare tutta la larghezza del pannello nel Task Tracker e nel Design Tracker.
  • Icona nella system tray visualizzata correttamente su Windows 11 (DPI awareness + conversione RGBA).
  • Il pin sulla barra delle applicazioni avvia correttamente Memento anziché una finestra flet vuota.
